## API


### **Products**
    

- _Get Products List_
   
        GET products/

- _Get Specific Product_
    
      GET products/{id}

- _Create Product_
   
      POST products/

- _Update Product_
    
        PUT products/{id}

- _Partial Update_
   
        PATCH products/{id}



### **Orders**

- _Get Orders List_

- _Create Order_

- _Update Order_

- _Partial Update_
